Project Name: WorkSpring Freelance Marketplace 

Live Link: https://workspring.netlify.app/

In this app you can: 

- Create user and store their information
- Register user by name, cover image, email, password
- Also direct login user by google authentication sign in
- Logout user
- Login user by email and password
- Update your profile name and photo
- Get all the job posted by other user and also yourself
- Add jobs by name, title, cover image, postedBy name, email, post date & time
- Update job that posted by yourself
- Read all jobs that posted by others and yourself
- Read details in job details page
- Accept job post by other user 
- Delete job that posted by yourself
- Accept job that posted by others
- See the list of your posted jobs
- See the accepted job listed selected by yourself also delete and cancel them 
- Light and Dark mode
